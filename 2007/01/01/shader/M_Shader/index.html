  • sample
    • float depth = SAMPLE_DEPTH_TEXTURE_PROJ(_CameraDepthTexture, UNITY_PROJ_COORD(i.projPos))
    • float sceneZ = LinearEyeDepth (depth)
    • float vertZ = i.projPos.z
    • float fade = vertZ - sceneZ
  • func
    • LinearEyeDepth
      • Z buffer to linear depth
      • 1.0 / (_ZBufferParams.z * z + _ZBufferParams.w)
    • Linear01Depth
      • Z buffer to linear 0..1 depth
      • 1.0 / (_ZBufferParams.x * z + _ZBufferParams.y)
  • work well in built-in and urp

Shader Variant

  • Toggle
    • 在编辑器UI上绘制Toggle，并且将其与multi_compile关联。定义float属性，标为Toggle，与其关联的multi_compile的关键字是属性名的全大写形式加上_ON
    • [Toggle(_MASKABLE_ON)] _Maskable (“Maskable?”, Float) = 1
    • #pragma multi_compile _ _MASKABLE_ON
    • https://aillieo.cn/post/2018-10-16-unity-3d-shader-notes-02/
  • KeywordEnum
    • https://www.jianshu.com/p/e2960e494d15
    • [KeywordEnum(None, Add, Multiply)] _Overlay(“Overlay mode”, int) = 0
    • 使用_Overlay一个变量来定义_OVERLAY_NONE，_OVERLAY_ADD， _OVERLAY_MULTIPLY三个关键字的互斥使用。
  • shader_feature，在游戏中没有使用的关键字，在构建游戏时不会生成对应的shader变体
  • 在运行时才用代码去开启和关闭关键字（Enable/DisableKeyword）的话，需要使用multi_compile
